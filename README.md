# 🚇 Metro Rail DSA Application

A fully functional metro rail routing system built using Data Structures & Algorithms from scratch.  

## Algorithms Used
- Dijkstra's Algorithm — shortest time route
- BFS — fewest stops route  
- A* Search — heuristic fastest route
- Floyd-Warshall — offline all-pairs cache
- Trie — station autocomplete search
- Segment Tree — zone-based fare queries

## How to Run
Just open `metro_rail_dsa_app.html` in any browser. No install needed.

## Features
- Route finder with 3 algorithm modes
- Interactive network map (Canvas)
- Fare calculator with zone matrix
- Live timetable
- DSA engine inspector
